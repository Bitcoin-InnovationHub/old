--- title: Ressources heading: Contribute projects to the Bitcoin InnovationHub description: Projects will be implemented in this website by adding them to this GitHub Repository.
  1. On GitHub, navigate to this site's repository.
  2. Navigate to the _posts directory.
  3. Create a new file called YYYY-MM-DD-NAME-OF-POST.md, replacing YYYY-MM-DD with the date of your post and NAME-OF-POST with the name of your post.
  4. Add the following YAML frontmatter to the top of the file, replacing POST TITLE with the post's title, YYYY-MM-DD hh:mm:ss -0000 with the date and time for the post, and CATEGORY-1 and CATEGORY-2 with as many categories you want for your post. ``` layout: post title: "POST TITLE" date: YYYY-MM-DD hh:mm:ss -0000 categories: CATEGORY-1 CATEGORY-2 ```
  5. Below the frontmatter, add content for your post.
